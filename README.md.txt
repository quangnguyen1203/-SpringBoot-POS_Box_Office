

# Ứng dụng điểm bán vé tại quầy của một rạp phim 

## Mục đích:
	- Xây dựng một app quản lý phù hợp với khả năng của các thành viên trong nhóm với đầy đủ các chức năng, nghiệp vụ của một ứng dụng bán hàng.
	- Củng cố kiến thức và rèn luyện kỹ năng làm việc nhóm.

# Môi trường và các công cụ sử dụng
	- Sử dụng phần mềm : IntelliJ IDEA
	- Framework sử dụng Spring Framework: Spring Boot, Spring Data JPA, Spring Security...

# Cài đặt sử dụng
	- Xem Backlog dự án tại [Link Backlog]( https://docs.google.com/spreadsheets/d/1dJelmdO3PZoXdaLiLLX3HqqwrkXKxIWQ-iaJRme2cho/edit#gid=0)
	- Clone dự án theo  [Link Github](https://github.com/quangnguyen1203/Spring_Boot_POS_Box_Office)
	- Cấu hình liên kết với database tại mục ' application.properties '
	- Import dữ liệu với file db.sql vào hệ quản trị cơ sở dữ liệu.
	- Tài khoản trang Admin: User đăng nhập: admin - Mật khẩu: 12345.
	- Tài khoản trang Nhân viên kỹ thuật: User đăng nhập: projection- Mật khẩu: 12345.
	- Tài khoản trang Nhân viên bán hàng: User đăng nhập: staff- Mật khẩu: 12345.
